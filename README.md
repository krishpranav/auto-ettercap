# auto-ettercap
A simple python automation for ettercap

# Steps To Run This Tool On Linux:
- bash install.sh
- git clone https://github.com/krishpranav/auto-ettercap
- cd auto-ettercap
- sudo chmod +x *
- python3 auto-ettercap.py

# Steps To Run This Tool On Mac:
- brew install ettercap
- git clone https://github.com/krishpranav/auto-ettercap
- cd auto-ettercap
- sudo chmod +x *
- python3 auto-ettercap.py
